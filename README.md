# Pomodoro Timer

A simple pomodoro timer app that runs in the browser.

---

## How to run it

You don't need to install anything special. Just follow these steps:

1. Download or clone this project to the computer
2. Open the project folder
3. Double-click the file called `index.html`
4. It will open in the browser and it's ready to use

That's it. No npm, no server, no setup needed.

---

## What it does

- Shows a countdown timer (25 minutes focus, 5 minutes break by default)
- You can change the focus and break minutes in the settings
- Has Start, Pause, and Reset buttons
- Plays a small beep sound when a session ends
- Automatically switches between focus and break
- Saves the completed focus sessions for today
- If you close and reopen the page, today's history is still there
- History resets automatically when a new day starts

---

## How to use it

1. Open `index.html` in the browser
2. Press **Start** to begin a focus session
3. Press **Pause** if you need to stop for a moment
4. Press **Start** again to resume
5. Press **Reset** to go back to the beginning
6. Change the numbers in the settings boxes to set the own focus or break time
